# ConnorWardell.com Website

Personal website for Connor Wardell, hosted on GitHub Pages at connorwardell.com.

## Purpose

A simple front page that:
- Displays "Connor Wardell's Website" as the main heading
- Links to [Dacite.dev](https://dacite.dev) with an explanation that it is where Connor's various developed and in-progress projects live

## Tech Stack

- Static site hosted on GitHub Pages
- HTML/CSS/JS

## Deployment

Deployed via GitHub Pages. The site is served from the repository root (or `docs/` folder depending on config) at connorwardell.com.
